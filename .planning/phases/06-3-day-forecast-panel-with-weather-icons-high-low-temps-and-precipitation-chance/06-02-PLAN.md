**Layout — RowLayout with Repeater for 3 day columns:**

```qml
RowLayout {
    anchors.fill: parent
    anchors.margins: 6
    spacing: 4

    Repeater {
        model: root.forecastData

        delegate: Item {
            Layout.fillWidth: true
            Layout.fillHeight: true

            readonly property var day: modelData

            Column {
                anchors.centerIn: parent
                spacing: 2

                // Weather icon
                Image {
                    anchors.horizontalCenter: parent.horizontalCenter
                    width: Math.min(parent.parent.width, parent.parent.height) * 0.4
                    height: width
                    source: root.iconPath(day.iconCode || "")
                    sourceSize.width: 64
                    sourceSize.height: 64
                    fillMode: Image.PreserveAspectFit
                }

                // High/Low temp on one line
                Row {
                    anchors.horizontalCenter: parent.horizontalCenter
                    spacing: 1
                    Text {
                        text: day.high > -998 ? day.high + "\u00B0" : "--\u00B0"
                        color: "#C84040"
                        font.pixelSize: Math.min(root.width, root.height) * 0.09
                        font.bold: true
                    }
                    Text {
                        text: "/"
                        color: "#C8A000"
                        font.pixelSize: Math.min(root.width, root.height) * 0.09
                    }
                    Text {
                        text: day.low > -998 ? day.low + "\u00B0" : "--\u00B0"
                        color: "#5B8DD9"
                        font.pixelSize: Math.min(root.width, root.height) * 0.09
                        font.bold: true
                    }
                }

                // Precipitation chance
                Text {
                    anchors.horizontalCenter: parent.horizontalCenter
                    text: day.precip + "%"
                    color: "#C8A000"
                    font.pixelSize: Math.min(root.width, root.height) * 0.07
                }
            }
        }
    }
}
```

Close the Rectangle with `}`.

**Design notes per user decisions:**
- No header label or title (maximize space for content)
- No day name labels (position implies today/tomorrow/day-after)
- High temp red (#C84040), low temp blue (#5B8DD9) — both already in DashboardGrid color palette
- Precipitation as plain percentage, no droplet decoration
- Same visual styling as all cells: #222222 background, rounded corners, #2A2A2A border
- sourceSize 64x64 prevents blurry SVG rasterization per Qt docs
