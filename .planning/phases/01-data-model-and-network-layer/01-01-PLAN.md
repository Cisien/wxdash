- `find_package(Qt6 REQUIRED COMPONENTS Core Network Test)`
- `qt_standard_project_setup()`
- `add_subdirectory(src)` and `add_subdirectory(tests)`
- `enable_testing()`

**src/CMakeLists.txt:**
- Define a library target `wxdash_lib` (STATIC) containing model and network source files. This allows tests to link against the library without building the executable.
- `qt_add_library(wxdash_lib STATIC)` with source files added as they're created
- For now, add `models/WeatherReadings.h` and `network/JsonParser.h`, `network/JsonParser.cpp`
- `target_link_libraries(wxdash_lib PUBLIC Qt6::Core Qt6::Network)`
- `target_include_directories(wxdash_lib PUBLIC ${CMAKE_CURRENT_SOURCE_DIR})`
- Define executable: `qt_add_executable(wxdash main.cpp)`, `target_link_libraries(wxdash PRIVATE wxdash_lib)`

**src/main.cpp:**
- Minimal QCoreApplication (no UI in Phase 1) that starts the event loop
- Just `QCoreApplication app(argc, argv); return app.exec();` for now — wiring comes in Plan 03

**tests/CMakeLists.txt:**

- Define test executable: `qt_add_executable(tst_JsonParser tst_JsonParser.cpp)`
- `target_link_libraries(tst_JsonParser PRIVATE wxdash_lib Qt6::Test)`
